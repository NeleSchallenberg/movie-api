Achievement 2 Project

Movie API

This project is the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL Endpoints

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a movie, containing a title, description, genre, year, and director property. Example:
{ "Title": "Drive My Car", "year": 2021, "director": "Ryusuke Hamaguchi" }
Get data about a genre by name /movies/genres/[name] GET None A JSON object holding data about a genre, containing a name and description property. Example:
{ "Name": "Drama", "Description": "In film and television, drama is a category or genre of narrative fiction..." }
Get data about a director by name /movies/directors/[name] GET None A JSON object holding data about a director, containing a name and bio property. Example:
{ "Name": "Wes Anderson", "Bio": "Wesley Wales Anderson was born on May 1, 1969, in Houston, Texas. His films are known for..." }
Register new users /users POST A JSON object holding data about the user to register, structured like: { "name": "Karen", "favoriteMovies": [] } A JSON object holding data about the user that was registered, including an ID: { "name": "Karen", "favoriteMovies": [], "id": "3681d316-902e-4921-9208-2ceba39b41a6" }
Update a user name /users/[id] PUT A JSON object holding data with the new name, structured like: { "name": "Regina Updated" } A JSON object holding updated data about the user. Example: { "id": 1, "name": "Regina Updated", "favoriteMovies": [ "The Prestige" ] }
Add a movie to a user's favorite list /users/[name]/[title] POST None A text message indicating whether the movie was successfully added to the list
Remove a movie from a user's favorite list /users/[name]/[title] DELETE None A text message indicating whether the movie was successfully removed from the list
Remove a user by ID /users/[id] DELETE None A text message indicating whether the user was successfully removed
